This module summarizes Kubernetes custom resources (CRs) used in the *AI Computer Vision at the Edge* Validated Pattern. Helm templates under `charts/all/` are the source of truth; examples below are simplified for workshop reading.

== Red Hat Connectivity Link (Kuadrant)

=== OIDCPolicy

*Purpose*: OIDC authentication on Gateway API HTTPRoutes.

*Product*: link:https://docs.redhat.com/en/documentation/red_hat_connectivity_link[Red Hat Connectivity Link]

[source,yaml]
----
apiVersion: extensions.kuadrant.io/v1alpha1
kind: OIDCPolicy
metadata:
  name: oidc-cv
  namespace: neuroface-gateway-system
spec:
  provider:
    issuerURL: https://sso.%HUB_DOMAIN%/realms/cv
    clientID: client-cv-user1
  targetRef:
    group: gateway.networking.k8s.io
    kind: HTTPRoute
    name: neuroface-cv-lb
----

*Connection*: OIDCPolicy → HTTPRoute → Gateway → backend Service (via Skupper on spokes).

=== AuthPolicy

*Purpose*: API key extraction and authorization (APIKEY header).

[source,yaml]
----
apiVersion: kuadrant.io/v1
kind: AuthPolicy
metadata:
  name: workshop-httpbin-auth
spec:
  targetRef:
    group: gateway.networking.k8s.io
    kind: HTTPRoute
    name: workshop-httpbin
  authRules:
    apikey:
      apiKeySelector:
        selector:
          matchLabels:
            kuadrant.io/auth-policy: workshop-httpbin
----

=== RateLimitPolicy

[source,yaml]
----
apiVersion: kuadrant.io/v1
kind: RateLimitPolicy
metadata:
  name: neuroface-cv-ratelimit
spec:
  targetRef:
    group: gateway.networking.k8s.io
    kind: HTTPRoute
    name: neuroface-cv-lb
  rateLimits:
    - limits:
        - maxValue: 120
          unit: minute
----

=== APIProduct

*Purpose*: Developer portal API catalog entry for Kuadrant plugin.

[source,yaml]
----
apiVersion: devportal.kuadrant.io/v1beta1
kind: APIProduct
metadata:
  name: workshop-httpbin
spec:
  reference:
    group: gateway.networking.k8s.io
    kind: HTTPRoute
    name: workshop-httpbin
----

== Kubernetes Gateway API

=== Gateway

*Docs*: link:https://gateway-api.sigs.k8s.io/[Gateway API]

[source,yaml]
----
apiVersion: gateway.networking.k8s.io/v1
kind: Gateway
metadata:
  name: neuroface-gateway
  namespace: neuroface-gateway-system
spec:
  gatewayClassName: istio
  listeners:
    - name: http
      port: 8080
      protocol: HTTP
----

=== HTTPRoute

[source,yaml]
----
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: neuroface-app-lb
spec:
  parentRefs:
    - name: neuroface-gateway
  rules:
    - backendRefs:
        - name: neuroface-app-east
          port: 8080
          weight: 50
        - name: neuroface-app-west
          port: 8080
          weight: 50
----

== Red Hat Build of Keycloak (RHBK)

=== KeycloakRealmImport

*Docs*: link:https://docs.redhat.com/en/documentation/red_hat_build_of_keycloak[RHBK documentation]

[source,yaml]
----
apiVersion: k8s.keycloak.org/v2alpha1
kind: KeycloakRealmImport
metadata:
  name: realm-cv
spec:
  keycloakCRName: keycloak
  realm:
    realm: cv
    users:
      - username: user1
        credentials:
          - type: password
            value: Welcome123!
----

== External Secrets Operator

[source,yaml]
----
apiVersion: external-secrets.io/v1

  name: keycloak-client-cv-user1

  secretStoreRef:
    name: vault-backend
    kind: ClusterSecretStore

=== Listener / Connector

[source,yaml]
----
apiVersion: skupper.io/v2alpha1
kind: Listener
metadata:
  name: neuroface-app-east
spec:
  routingKey: neuroface-app-east
  port: 8080
----
apiVersion: skupper.io/v2alpha1
kind: Connector
metadata:
  name: neuroface-app-east
spec:
  routingKey: neuroface-app-east
  selector: app=neuroface-backend
  port: 8080
----
