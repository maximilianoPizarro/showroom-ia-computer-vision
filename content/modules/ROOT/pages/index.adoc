Welcome to the **AI Computer Vision at the Edge** workshop — a focused 1-hour journey through multi-cluster AI inference.

In 60 minutes you deploy a personal NeuroFace instance from Developer Hub, explore OpenShift AI MaaS connections, discover RHCL Gateway objects, and test inference with curl.

== What you will do

* Deploy NeuroFace from the *AI Computer Vision at the Edge* Developer Hub template
* Explore OpenShift AI `maas-workshop` model connections
* Inspect Gateway API, HTTPRoute, and Skupper ExternalName Services
* Obtain a MaaS API key and test chat completion with curl
* Review GitOps configuration you can replicate in future patterns

== Workshop access

Use your assigned workshop user (`user1` through `user30`) with password `Welcome123!` to log in to:

* OCP Console (HTPasswd identity provider `workshop-users`)
* Red Hat Developer Hub (Keycloak realm `backstage`)
* GitLab and DevSpaces (provisioned per user)

== Lab terminal

The embedded terminal on the right supports helper commands:

[source,sh,role=execute]
----
hub-login user1
east-login user1
west-login user1
neuroface-cv-status
neuroface-cv-traffic
neuroface-spoke-status
----

TIP: Replace `user1` with your assigned user number.
